# Screenshots

Drop PNG screenshots here using these exact filenames so they render in the main `README.md`:

| Filename | Page to capture | URL |
|----------|-----------------|-----|
| `dashboard.png` | Executive Dashboard (KPI cards + charts) | http://localhost:3000/dashboard |
| `findings.png` | Findings list, then open a finding to show the AI analysis + mapped controls | http://localhost:3000/findings |
| `mapping.png` | Mapping Engine result for e.g. "Firewall Disabled" | http://localhost:3000/mapping |
| `compliance.png` | Compliance scoring gauge + per-family bars | http://localhost:3000/compliance |

Optional extras you can also add: `risks.png`, `remediation.png`, `controls.png`, `login.png`.

## How to capture (Windows)

1. Make sure both servers are running and you're logged in as **Admin**.
2. Open the page in your browser at the URL above.
3. Use the Windows Snipping Tool (`Win` + `Shift` + `S`), select the browser content area, and save as the filename above into this folder.
4. For full‑width, crisp shots, maximize the browser window (1440px+ wide) before capturing.

Tip: dark, full‑page captures of the Dashboard and the Findings detail panel make the strongest first impression for recruiters.
